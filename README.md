# Deploy ROS Noetic on Cloud (AWS EC2)

## Description
This project demonstrates the deployment of ROS Noetic on AWS EC2 using Docker.
It allows developers to run robotic applications and simulations on the cloud.

## Technologies Used
- AWS EC2
- Ubuntu
- Docker
- ROS Noetic

## Steps Followed
1. Created EC2 instance on AWS
2. Installed Docker on Ubuntu
3. Pulled ROS Noetic Docker image
4. Ran ROS container
5. Verified using roscore

## Result
ROS Noetic successfully deployed and running on cloud infrastructure.
